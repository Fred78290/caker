🍰 Caker - Outil de Gestion de Machines Virtuelles

Ce programme d'installation configurera Caker globalement sur votre système macOS.
Caker est un outil moderne basé sur Swift pour construire et gérer des machines virtuelles avec le framework de Virtualisation d'Apple.

🚀 Composants Installés

Caker.app - Application Bureau macOS

Interface utilisateur moderne pour la gestion quotidienne des VM :

  • Installation et gestion du service caked
  • Création et configuration de machines virtuelles
  • Interface VNC intégrée
  • Surveillance en temps réel
  • Diagnostics et journaux accessibles

caked - Démon Principal

Le service démon qui gère le cycle de vie des machines virtuelles :

  • Démarrage, arrêt et surveillance des VM
  • Gestion des ressources (CPU, mémoire, stockage)
  • Enregistrement du service système
  • Surveillance de la santé et récupération automatique

cakectl - Interface en Ligne de Commande

Outil de contrôle pour interagir avec caked :

  • Construction et déploiement d'applications
  • Gestion des configurations VM
  • Visualisation des journaux et du statut
  • Contrôle des processus démon

✨ Fonctionnalités Clés

  • Redirection de Ports : Sockets TCP ou Unix entre VM et hôte
  • Gestion Réseau : Modes bridge, hébergé ou NAT
  • Cloud-Init : Support pour l'initialisation des VM
  • Agent Intégré : Installation automatique d'agent dans les VM
  • Configuration Dynamique : Mises à jour en temps réel

🛠 Premiers Pas

Après installation, vous pouvez commencer à utiliser Caker :

Application Bureau :

Lancez Caker.app depuis le dossier Applications ou Spotlight.

Installer le service :

caked service install

Interface CLI :

cakectl --help
