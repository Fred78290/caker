🍰 Caker - Virtual Machine Management Tool

This installer will configure Caker globally on your macOS system.
Caker is a modern Swift-based tool for building and managing virtual machines with Apple's Virtualization framework.

🚀 Installed Components

Caker.app - macOS Desktop Application

Modern user interface for daily VM management:

  • Install and manage caked service
  • Creating and configuring virtual machines
  • Integrated VNC interface
  • Real-time monitoring
  • Accessible diagnostics and logs

caked - Core Daemon

The daemon service that manages virtual machine lifecycle:

  • Starting, stopping, and monitoring VMs
  • Resource management (CPU, memory, storage)
  • System service registration
  • Health monitoring and auto-recovery

cakectl - Command Line Interface

Control tool for interacting with caked:

  • Building and deploying applications
  • Managing VM configurations
  • Viewing logs and status
  • Controlling daemon processes

✨ Key Features

  • Port Forwarding: TCP or Unix sockets between VM and host
  • Network Management: Bridge, hosted, or NAT modes
  • Cloud-Init: Support for VM initialization
  • Integrated Agent: Automatic agent installation in VMs
  • Dynamic Configuration: Real-time updates

🛠 Getting Started

After installation, you can start using Caker:

Desktop Application:

Launch Caker.app from Applications folder or Spotlight.

Install the service:

caked service install

CLI Interface:

cakectl --help
